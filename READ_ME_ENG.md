This program is in the VBA programming language,
finds and displays the sum of the longest along the increasing sequence of successive elements.
If there are several such increasing sequences, then any of them can be derived.
The main module Module1 contains the following main functions corresponding to the main buttons on the form:
1) button to fill the array - it fills the elements of the array with random positive values;
2) button to display the amount - finds and displays the amount of the longest in the increasing sequence
consecutive elements, if there are several such increasing sequences, then it outputs any of them;
3) exit button - closes the form.
